# Aritra Basu - Photographer Portfolio Website

A modern, responsive photographer portfolio website built with HTML, CSS, and JavaScript.

## Features

- 🎨 **Responsive Design** - Works perfectly on all devices (mobile, tablet, desktop)
- 📸 **Portfolio Gallery** - Filterable gallery with multiple photography categories
- 📝 **Blog Section** - Share photography tips and insights
- 💬 **Contact Form** - Easy way for clients to get in touch
- ✨ **Smooth Animations** - Beautiful transitions and interactions
- 📱 **Mobile Menu** - Hamburger menu for better mobile navigation

## Sections

1. **Hero** - Eye-catching landing section with call-to-action
2. **Portfolio** - Filterable gallery showcasing diverse photography work
3. **About** - Personal biography and photography philosophy
4. **Blog** - Photography tips and insights
5. **Contact** - Contact form for client inquiries
6. **Footer** - Quick navigation and social media links

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Unsplash API (for random images)

## How to Use

1. Clone this repository:
   ```bash
   git clone https://github.com/aritra1311/Web.git
   cd Web
   ```

2. Open `index.html` in your web browser, or run a local server:
   ```bash
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000`

## Customization

- **Images** - Replace placeholder images with your own photos
- **Text** - Update content in `index.html`
- **Colors** - Modify CSS variables in `style.css`:
  - `--primary-color`: Main dark color
  - `--secondary-color`: Accent color
  - Update other colors as needed
